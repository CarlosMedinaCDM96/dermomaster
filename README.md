# Dermo Máster - Plataforma de Formación Médica

Proyecto final de maquetación web enfocado en el sector sanitario.

## Tecnologías utilizadas
- HTML5 Semántico (Estructura)
- CSS3 (Flexbox, Grid, Variables, BEM)
- Vanilla JavaScript (Interacciones y validación)
- Formatos WebP para optimización de rendimiento

## Créditos
- **Diseño de Identidad:** Elaboración propia (Logo y Branding).
- **Tipografías:** Montserrat e Inter (Google Fonts).
- **Iconografía:** Phosphor Icons (Licencia MIT).
